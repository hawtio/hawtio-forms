In this case we're passing a 'selectors' property for the FormElement, hawtio-forms2 will do a jquery 'find' on the control element for each selector instance and pass the result of 'find' into the selector function provided here. This will happen before interpolation and compilation of the entire form template, so it's possible to add Angular directives on the fly etc. Use 'el' to get the entire form-group for that element. In this example, changing the 'select' value to anything other than 'Two' will hide the 'Name' element.
